# Terminal

### Installer ESLint

```
npm install --g eslint
```

### Initialiser ESLint
```
npm init @eslint/config
```

### Vérifier fichier avec ESLint
```
.\node_modules\.bin\eslint (nom_fichier.js)
```

### Corriger fichier avec ESLint
```
.\node_modules\.bin\eslint (nom_fichier.js) --fix
```

### Créer Doc Fichier avec JSDoc
```
.\node_modules\.bin\jsdoc (nom_fichier.js)
```
