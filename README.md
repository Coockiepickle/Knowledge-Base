# My Knowledge Base

## 📚 Overview

This repository contains my personal technical knowledge base, focused on networking, infrastructure, DevOps and IT project work.  
It centralizes my notes, step‑by‑step guides, configurations and reference sheets built over the years.

> [!IMPORTANT]
> All documentation is written by me, close collaborators or people who explicitly allowed me to publish their work (sometimes with help from AI tools).  
> See the [LICENSE](LICENSE.md) for details before reusing any content.


## 🧭 What’s inside

Some of the main topics covered:

- Networking (VLANs, VPNs, routing, firewalls, etc.)
- Infrastructure as Code (Ansible, Terraform, automated deployments)
- Containers & Kubernetes (Docker, k8s, troubleshooting)
- General tooling and workflows (MkDocs, CI/CD, monitoring, etc.)
